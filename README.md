# Email Categorizer

A Python automation tool that categorizes volunteer application emails, inspired by manual workflow observations at Toronto Animal Services.

## Overview

This project addresses a real workflow inefficiency: manually sorting hundreds of volunteer application emails. The tool automates categorization using keyword analysis and provides reporting capabilities.

## Features

- **Keyword-based Categorization:** Automatically sorts emails into categories (DOG_FOSTER, CAT_FOSTER, etc.)
- **Batch Processing:** Processes entire directories of email files
- **Multiple Interfaces:** 
  - Command-line interface for automation
  - Web interface (Flask) for easy use
- **Reporting:** Generates CSV reports with statistics
